Privacy & Data Protection

Privacy Policy

This Privacy Policy explains how Smart Application Tracker collects, uses, and protects user information while using the platform and its authentication services.

Last updated: May 2026

1. Introduction

Smart Application Tracker is a web application designed to help users organize, track, and manage job and internship applications efficiently.

By using this application, you agree to the collection and use of information in accordance with this Privacy Policy.

2. Information We Collect

The application may collect the following information:

  • Username and email address
  • Authentication information
  • Job application details entered by users
  • Profile information from Google OAuth login
  • Session and login activity

3. Authentication & Google Sign-In

Users may choose to authenticate using Google OAuth.

When signing in with Google, the application may access:

  • Email address
  • Basic profile information
  • Profile image

Passwords associated with Google accounts are never stored by this application.

4. Password Security

Passwords created through manual registration are securely hashed before storage using industry-standard security practices.

Plain-text passwords are never stored in the database.

5. How Information Is Used

User information is used to:

  • Create and manage accounts
  • Store job application data
  • Provide analytics and tracking features
  • Enable secure authentication
  • Support password recovery functionality
  • Improve application experience and reliability

6. Cookies & Sessions

Smart Application Tracker may use secure cookies and sessions to maintain authentication states and improve user experience.

These technologies help users remain logged in securely across browsing sessions.

7. Third-Party Services

The application may rely on trusted third-party services, including:

  • Google OAuth authentication
  • Flask-Mail for password reset emails
  • Cloud hosting providers for deployment

These services may process limited user information as part of their functionality.

8. Data Storage

Application data is stored securely using database systems configured for the project environment.

Reasonable efforts are made to protect user information against unauthorized access or misuse.

9. Educational & Portfolio Project

Smart Application Tracker is currently developed as an educational and portfolio project.

Features and policies may evolve over time as the application continues to improve.

10. Contact

For questions regarding this Privacy Policy or the application, please contact the developer through the project's GitHub repository.
